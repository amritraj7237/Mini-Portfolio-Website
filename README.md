# Mini-Portfolio-Website --Amrit Rai
 
A personal portfolio website built using **pure HTML** (no CSS, no JavaScript) to showcase my skills, projects, education, and contact information.
 
---
 
## About the Project
 
This is my first self-made portfolio website. I learned HTML and built this entirely on my own in about **1 day**. It covers all the essential sections of a personal portfolio — from an About Me section to my ML projects.
 
---

## Live Preview
 
 **Deployed Site:** [https://amritraj7237.github.io/Mini-Portfolio-Website/](https://amritraj7237.github.io/Mini-Portfolio-Website/)


---
 
## Sections Included
 
| Section      | Description                                      |
|--------------|--------------------------------------------------|
| **Header**   | Name and title banner                            |
| **Navigation** | Anchor links to all sections                  |
| **About Me** | Personal info, photo, hobbies, and introduction  |
| **My Skills**| Programming languages, libraries & ML algorithms |
| **Projects** | Details of completed projects with links         |
| **Education**| School and college details in a table format     |
| **Contact**  | Email, phone, LinkedIn, GitHub, and address      |
 
---
 
## Technologies Used
 
- **HTML** only (no CSS, no JavaScript)
- Tables for layout
- Anchor tags for navigation
- `<font>`, `bgcolor`, `border` attributes for styling
